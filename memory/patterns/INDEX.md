# Patterns — Extracted Craft Wisdom

Patterns are discovered automatically through the writing cycle. Every scene review extracts what works.

## Confidence Levels

| Level | Meaning | When |
|-------|---------|------|
| **LOW** | Experimental, unvalidated | First time a pattern appears |
| **MEDIUM** | Promising, likely reliable | Pattern confirmed in a second scene |
| **HIGH** | Validated, highly reliable | Used 3+ times with consistent results |

HIGH confidence patterns are automatically surfaced when you write new scenes.

## Pattern Files

- `prose-patterns.md` — Sentence structures, description techniques, transitions
- `dialogue-patterns.md` — Speech rhythms, subtext methods, tag strategies
- `pacing-patterns.md` — Scene rhythm, tension escalation, paragraph-length effects

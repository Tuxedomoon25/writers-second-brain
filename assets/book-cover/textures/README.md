# Cloth & Paper Textures

Reusable texture assets for book cover backgrounds. Generate on first use via Midjourney or PS Generative Fill, then save here for consistency.

## Expected Files

- `cloth-sage-green.png` — Sage green linen cloth (SotB)
- `cloth-gold.png` — Muted gold/tan cloth (The Great Story)
- `cloth-navy.png` — Navy/slate blue cloth (The Captain's Log)
- `paper-grain.png` — Subtle vintage paper grain overlay (cream tones)
- `paper-grain-dark.png` — Paper grain for dark backgrounds

## Specs

- Resolution: 1200x1800px minimum (will be scaled to cover dimensions)
- Format: PNG, seamless/tileable preferred
- Color mode: RGB (converted to CMYK at export time)

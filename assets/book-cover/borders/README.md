# Series Border Frames

Decorative border/frame assets for book series covers.

## Expected Files

- `ghent-cycle-frame.png` — Gold Celtic knotwork border for The Ghent Cycle (SotB series)

## Specs

- Resolution: Match cover dimensions (transparent PNG)
- Format: PNG with transparency
- Color: Gold/cream elements on transparent background
- Usage: Set blend mode to SCREEN or LIGHTEN over dark backgrounds

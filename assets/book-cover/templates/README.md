# KDP & IngramSpark Cover Templates

Download platform-specific cover templates here for alignment verification.

## How to Get Templates

- **KDP**: Download from https://kdp.amazon.com/en_US/cover-calculator
- **IngramSpark**: Generated during title setup (PDF template with all guides)

## Usage

Place template as the top layer in Photoshop, set to Multiply blend mode at 50% opacity. Toggle visibility to check alignment of bleed, safe zone, spine, and barcode area.

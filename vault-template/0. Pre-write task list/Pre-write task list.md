# Pre-Write Task List

Complete these before writing your first scene.

## Foundation

- [ ] Run `/setup` — Define your voice, rules, and quality thresholds
- [ ] Run `/story-forge` — Build your concept, outline, and beat sheet
- [ ] Run `/character-forge` for your protagonist
- [ ] Run `/character-forge` for your antagonist
- [ ] Fill in at least one Setting profile (`5. Settings/`)

## Outline

- [ ] Complete Act 1 scene cards (`2. Outlines/Scene Cards/`)
- [ ] Write deep-dive beats for Chapter 1 (`2. Outlines/Deep Dive Beats/`)
- [ ] Define the Master Tone of Voice (`2. Outlines/Tone of voice/`)

## Ready to Write

- [ ] First scene identified and outlined
- [ ] POV character profile complete
- [ ] Setting for first scene described
- [ ] Run `/write-scene` for your first scene
